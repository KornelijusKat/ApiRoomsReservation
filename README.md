Create config.env file at root. With these fields:

NODE_ENV=development
PORT=8000(your port)
# MongoDB Database Connection
DATABASE=mongodb+srv://<username>:<PASSWORD>@cluster0.rb8b8.mongodb.net/<dbname>(preferablyhotels>?retryWrites=true&w=majority&appName=Cluster0
DATABASE_PASSWORD=<your db Password>

To run npm i and then npm start to run application.
